Detailed Description

Represents the grid used to hold the velocity field, boundaries, density, etc. This holds one or two RenderTexture which allows the use of ping-pong rendering: one RenderTexture is used as input while the other is used as output, when we're done we swap the two. This is used for iterative algorithms such as the LinearSolvers. The equal operator is overloaded and works with the Operator class and makes for succint code when running a shader with multiple inputs.
